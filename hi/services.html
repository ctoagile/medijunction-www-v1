प्लेटफ़ॉर्म की क्षमताएँ

भारत भर में संसाधन-सीमित परिवेश में सामुदायिक स्वास्थ्य कार्यक्रमों को सशक्त बनाने के लिए डिज़ाइन किए गए उपकरणों का व्यापक सूट

सामुदायिक स्वास्थ्य समन्वय

स्थानीय महिलाओं को सामुदायिक स्वास्थ्य समन्वयक के रूप में प्रशिक्षित और सशक्त करें। हमारा प्लेटफ़ॉर्म शिक्षार्थी से वरिष्ठ समन्वयक तक संरचित भूमिका प्रगति प्रदान करता है, टियर-आधारित एक्सेस कंट्रोल और प्रति-कार्यक्रम संदर्भ-संवेदनशील रोल लेबल के साथ।

  • 6-स्तरीय रोल पदानुक्रम
  • i18n रोल लेबल (हिंदी/अंग्रेज़ी)
  • रोगी पंजीकरण के लिए इनटेक विज़ार्ड
  • एनालिटिक्स के साथ सामुदायिक डैशबोर्ड

टेलीकंसल्टेशन

Jitsi Meet के माध्यम से वीडियो परामर्श जो ग्रामीण रोगियों को विशेषज्ञ डॉक्टरों से जोड़ता है। सामुदायिक स्वास्थ्य समन्वयक वीडियो कॉल शुरू कर सकते हैं, रोगी इतिहास साझा कर सकते हैं और क्षेत्र से दूरस्थ परामर्श की सुविधा प्रदान कर सकते हैं।

  • एम्बेडेड Jitsi वीडियो कॉल
  • AI-संचालित केस तैयारी
  • एसिंक चैट परामर्श
  • डिजिटल प्रिस्क्रिप्शन जनरेशन

AI-सहायित ट्राइएज

Claude AI जोखिम मूल्यांकन सामुदायिक स्वास्थ्य कार्यकर्ताओं को रोगी के लक्षणों का मूल्यांकन करने और देखभाल को प्राथमिकता देने में मदद करता है। सिस्टम सभी सिफ़ारिशों के लिए मानव-इन-द-लूप सत्यापन बनाए रखते हुए सुसंगत आउटपुट के लिए संरचित टूल-उपयोग का उपयोग करता है।

  • मल्टी-प्रोवाइडर AI (Claude/Gemini/Groq)
  • संरचित जोखिम मूल्यांकन
  • केवल सहायक — कोई स्वायत्त निर्णय नहीं
  • रेफरल सिफ़ारिशें

डिजिटल स्वास्थ्य रिकॉर्ड

ABHA ID एकीकरण और FHIR R4 बंडल जनरेशन के साथ ABDM-अनुपालित डिजिटल स्वास्थ्य रिकॉर्ड। रोगी डेटा Fernet एन्क्रिप्शन का उपयोग करके आराम पर एन्क्रिप्ट किया जाता है, स्वास्थ्य सूचना विनिमय के माध्यम से सहमति-आधारित साझाकरण के साथ।

  • ABHA ID लिंकिंग
  • FHIR R4 स्वास्थ्य रिकॉर्ड
  • HIE-CM सहमति प्रबंधन
  • PII एन्क्रिप्शन (Fernet)

एकीकृत सूचनाएँ

व्यापक रोगी संचार के लिए WhatsApp Cloud API + Fast2SMS। सिस्टम उपलब्ध होने पर WhatsApp को प्राथमिकता देता है, ग्रामीण रोगियों के लिए SMS पर फ़ॉलबैक करता है, और मल्टी-टेनेंट तैनाती के लिए प्रति-कार्यक्रम क्रेडेंशियल का समर्थन करता है।

  • WhatsApp Cloud API एकीकरण
  • DLT टेम्पलेट के साथ Fast2SMS
  • स्वचालित फ़ॉलो-अप रिमाइंडर
  • प्रति-कार्यक्रम क्रेडेंशियल

ऑफ़लाइन-फ़र्स्ट PWA

Dexie.js (IndexedDB) और Workbox सर्विस वर्कर्स के साथ बनाया गया प्रोग्रेसिव वेब ऐप। स्वास्थ्य कार्यकर्ता बिना इंटरनेट कनेक्टिविटी वाले क्षेत्रों में भी रोगियों का पंजीकरण, ट्राइएज और परामर्श प्रबंधन कर सकते हैं।

  • IndexedDB लोकल स्टोरेज
  • पुश-देन-पुल सिंक इंजन
  • मोबाइल पर इंस्टॉल करने योग्य
  • वेब पुश नोटिफ़िकेशन (VAPID)
तकनीकी स्टैक

आधुनिक, युद्ध-परीक्षित तकनीकों से निर्मित

हमारा प्लेटफ़ॉर्म विश्वसनीयता, प्रदर्शन और रखरखाव योग्यता के लिए सर्वश्रेष्ठ ओपन-सोर्स तकनीकों का लाभ उठाता है।

  • फ्रंटएंड: Next.js 14 (App Router) + Tailwind CSS + TypeScript
  • बैकएंड: Python FastAPI + SQLAlchemy Async + PostgreSQL 16

  • ऑफ़लाइन: Dexie.js (IndexedDB) + Workbox सर्विस वर्कर्स
  • डिप्लॉयमेंट: Docker + GitHub Actions CI/CD + Caddy
